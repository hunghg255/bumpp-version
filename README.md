## Install

```bash
npm i bumpp-version@latest -D
```

## Usage

```json
 "scripts": {
    "bump": "bumpp-version"
  },
```

## Run Script

```bash
npx bumpp-version@latest

pnpx bumpp-version@latest

bunx bumpp-version@latest
```
